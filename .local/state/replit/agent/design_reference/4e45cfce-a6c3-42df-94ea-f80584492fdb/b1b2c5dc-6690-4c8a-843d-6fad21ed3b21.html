Deployment Instructions

Step-by-step guide to deploy on Vercel

1

Repository Setup

Push the restructured code to your Git repository (GitHub, GitLab, or Bitbucket)

git add .
git commit -m "Restructure for Vercel deployment"
git push origin main
2

Import to Vercel

Import your repository to Vercel and configure the build settings

• Root Directory: ./
• Framework Preset: Other
• Build Command: cd frontend && npm run build
3

Configure Environment Variables

Add the required environment variables in Vercel's project settings

⚠️ Ensure DATABASE_URL points to your production PostgreSQL database
4

Deploy

Deploy your application and verify all functionality works correctly

✅ Frontend will be served from Vercel's global CDN
✅ Backend APIs will run as serverless functions
✅ Database connections will be pooled automatically
